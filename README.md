# 🧩 スライドパズルゲーム

画像を完成させるシンプルなスライドパズルゲームです。
ブラウザ上で遊べます。

---

## 🎮 遊び方

1. タイトル画面で難易度を選択します

   * **かんたん**：3×3
   * **ふつう**：4×4
   * **むずかしい**：5×5
2. 空白マスの上下左右にあるピースをクリックすると移動します
3. すべてのピースを正しい位置に並べるとクリアです 🎉

---

## 🔊 サウンドについて

* BGM：ページ遷移後も同じBGMが流れます
* 効果音：

  *難易度選択時
  * パズルのピースを動かした時
* 右上の 🔊 / 🔇 ボタンで BGM の ON / OFF を切り替えられます


---

## 💡 機能一覧

* 難易度選択（3段階）
* ヒント画像の表示 / 非表示
* BGM / 効果音（SE）
* クリア判定 & クリア画面遷移
* Xで共有
